1. History

LLVM Compiler Infrastucture

A collection of modular compiler and toolchain technologies.

Core technologies:

  • LLVM IR (intermediate representation)
    • machine independent assembly language
  • Optimizer and code generator for LLVM IR.
  • Clang (main compiler)

Frontend, Optimizer, Backend

(from the AOSA book)

Clang

A C language family frontend for LLVM.

  • Open source.
  • Developed by Google, Apple and others.
  • Compatible with GCC.
    • Less CPU and memory intensive at compiling.
    • Produces marginaly less optimal code.
  • Other advantages: C++11 complete, default in growing number of Unix OSes and Linux distros
